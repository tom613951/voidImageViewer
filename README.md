# void Image Viewer (中文版)
适用于 Windows 的轻量级图像查看器，支持 GIF/WEBP 动态图片。  
以最快速度打开并显示 BMP, GIF, ICO, PNG, JPG, TIF 和 WEBP 图像。  
精准还原并播放 GIF 和 WEBP 动图。
